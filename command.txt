2. git init

3. git add 
I will be using "git add ."
. = all changes of all of the files in this folder
name.type = all changes of the specific name.type file


4. Git config

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

will need to do them at least once in laptop,
to add myself as the author/user,
or if I want to change aurhor/user.

git config --global --list

this to check the current author/user


5. git commit -m "first commit"
this is a comment section, here we say what changes we made on this version update
kinda like update log.


6. git branch -M main
this is like adding or updating folder location? It's not clear yet.
but this will change or stuff, gotta wait and see.


7. git remote add origin https://github.com/El-dric/My-first-repository.git
this one sets the actual online github repository location, where the file will go and 
the changes would be saved.
When done first time in your PC, it's also a verification initializer. You will see.
if you try to do this twice in a same folder, this will show you ERROR and tell you,
the location already exists.

git remote -v
use this to check the current location


8. git push -u origin main
This is the litteral PUSH/LAUNCH button

git push
